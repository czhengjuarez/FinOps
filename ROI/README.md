# Design Investment ROI Calculator

A comprehensive web-based calculator for measuring the return on investment of design team resources, tools, and processes.

## Features

- **Design System ROI Calculator**: Calculate savings from implementing a design system
- **Tooling Investment Analysis**: Measure ROI of design tool investments
- **Headcount Planning**: Determine optimal team size and hiring needs
- **Budget Allocation Framework**: Plan design budget distribution across categories

## Installation

1. Download all files to your web server directory
2. Ensure the following file structure:
   ```
   /your-website-folder/
   ├── index.html
   ├── styles.css
   ├── script.js
   └── README.md
   ```
3. Open index.html in a web browser or upload to your web server

## File Structure

- `index.html` - Main calculator interface
- `styles.css` - All styling and responsive design
- `script.js` - Calculator functions and interactivity
- `README.md` - Documentation

## Customization

### Branding
- Update the footer links in `index.html` to point to your website
- Modify the color scheme in `styles.css` (search for color codes like #3b82f6)
- Add your logo by updating the header section

### Default Values
- Modify default input values in `index.html` to match your market/industry
- Update recommended percentages in the budget allocation section

### Calculations
- Customize calculation formulas in `script.js` if needed
- Add additional metrics or modify existing ones

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive design
- No external dependencies required
